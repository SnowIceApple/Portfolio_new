주요 요소들
  • GNB

    • nav에 hover시 header 높이가 변경되는 효과를 animate로 구현

    • 메뉴 닫기 버튼 구현

  • Layout

    • float, position 등으로 위치를 잡은 원 사이트와는 달리 flex로 구현

    • 원 사이트와 동일한 탭키 포커스 방향을 유지하기 위해 flex-direction: column 지정 후 컨테이너의 높이를 px단위로 설정하는 방식을 사용

  • Accessbility

    • 모달 버튼의 경우 모달의 열림/닫힘 여부에 따라 접근성 텍스트의 내용이 변경됨

    • 검색 모달의 검색하기 버튼에서 탭키를 누르면 위의 ir 텍스트의 내용이 변경된 버튼으로 포커스가 잡히게 되고, 만약 해당 버튼을 클릭하여 모달창을 닫지 않고 탭키를 누르면 모달에 포커스가 잡히게 됨

    • 탭키 만으로 이동 가능한 메인 GNB

    • 본문 등을 오갈 수 있는 스킵 네비게이션

    • 모달을 탭키로 탐색할 때 클로징 버튼이 포커스 되었음에도 클릭하지 않고 재차 탭키를 누르는 경우 모달을 벗어나지 않고 해당 모달의 컨텐츠를 포커스 하도록 설정
